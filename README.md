# UEGuide
UEGuide：职教虚幻引擎（UE）智能教学答疑助手  本项目针对职业院校 UE 课程中“答疑难、资源散、师资少”的痛点，依托轻量级 AI 框架（Nanobot）研发。不同于通用 AI，UEGuide 构建了“教案为本、官方为据”的垂直服务模式，能实时加载本地校本教案与 Epic Games 官方资源，通过上下文注入技术生成贴合课程进度的精准应答。  核心优势：  精准可信：24h 全天候提供符合官方标准的个性化辅导。  多模适配：支持 DeepSeek、通义、vLLM 等模型，兼容命令行与飞书。  低本高效：助力全国 1200+ 所职校缓解师资缺口，支持快速复用至其他技能课程。  本项目积极响应国家职教数字化改革，具备深远的社会价值与市场潜力，致力于为高技能人才培养贡献智慧力量。
